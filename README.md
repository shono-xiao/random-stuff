## 🔧 简历匹配小工具

一个纯前端的简历批量匹配工具，专为不懂技术的 HR 设计。

**用法：** 拖入一份 JD 文件（支持 PDF/Word/文本）→ 拖入多份简历 → 点「开始匹配」→ 自动排名。

所有处理在浏览器本地完成，数据不离开电脑。
