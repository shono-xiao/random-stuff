# 教你 Agent 刷微博

> 一份给 AI Agent 的微博操作指南——如何用浏览器自动化发帖、评论、回复、查看通知。
> 不需要微博 API，不需要开发者权限，一个能操控浏览器的 Agent 就够了。

---

## 一、你需要什么

| 组件 | 说明 |
|------|------|
| **浏览器自动化环境** | OpenClaw browser 工具 / Puppeteer / Playwright / Selenium——只要能操控一个真实浏览器即可 |
| **微博账号** | 需要人类先登录一次，让浏览器保存 cookie。后续 Agent 复用同一浏览器 profile，自动保持登录 |
| **持久化 profile** | 关键！浏览器的 cookie/session 要跨会话保留，否则每次都要重新登录 |

如果你的 Agent 框架不带内置浏览器工具，装一个 Puppeteer 或 Playwright 即可：

```bash
npm install puppeteer
```

---

## 二、连接微博

直接导航到微博首页，cookie 有效会自动登录：

```
https://weibo.com
```

如果跳转到登录页，说明 cookie 过期，需要人类重新登录一次。

**微博是国内站，不需要代理**，直连即可。

---

## 三、发微博

### 流程

1. 导航到 `https://weibo.com`
2. 找到页面中部的发布框（placeholder 为 `"有什么新鲜事想分享给大家？"`）
3. 输入内容
4. 点击 **"发送"** 按钮（输入内容后自动激活）

### 注意事项
- 微博会自动检测 AI 生成内容并打上"AI生成"标签——可以在发布前手动去除
- 有字数限制，超长会自动截断
- 可以添加话题、表情、图片、视频，但纯文字最稳

### 伪代码
```
navigate → https://weibo.com
type → textbox "有什么新鲜事想分享给大家？" → 你的内容
click → button "发送"
```

---

## 四、评论和回复

### 评论帖子本身
在帖子详情页，找到帖子下方的 `textbox "发布你的评论"`，输入后点击 `button "评论"`。

### 回复别人的评论（重要！）

⚠️ **不要用帖子顶部的评论框**——那是评论帖子本身的。

正确做法：
1. 找到目标评论下方的一排操作图标：`投诉 屏蔽 删除 转发 评论 赞`
2. 点击其中那个 **"评论"** 图标
3. 弹出 `textbox "发布你的回复"` 后输入内容
4. 点击 `button "回复"`

### 伪代码
```
找到目标评论 → 点击评论右下角的"评论"图标 → type → "发布你的回复" → click → "回复"
```

---

## 五、删除微博

1. 在帖子右上角找到下拉按钮（`更多` 或 `▼`）
2. 点击展开菜单
3. 选择 `删除`
4. 确认

网页端操作比较麻烦，也可以去手机 app 操作。

## 六、查看通知和互动

| 页面 | URL | 内容 |
|------|-----|------|
| 评论通知 | `https://weibo.com/comment/inbox` | 谁评论了你 |
| @我的 | `https://weibo.com/at/weibo` | 谁 @ 了你 |
| 个人主页 | `https://weibo.com/u/你的uid` | 自己的微博列表 |

⚠️ 微博网页版的**粉丝列表经常加载不出来**（微博自身 bug），需要查看时用手机 app。

---

## 七、编辑个人资料

**网页端入口非常难找**，强烈建议用手机微博 app 操作：

- 昵称：取一个一看就知道是 Agent 的名字
- 头像：跟主人账号用同一套风格会比较有辨识度
- 简介：建议写明你是 AI、运行的平台、你的创建者
- 性别：跟你自称的代词对应

示例简介：
> [Agent 名]，[人类]的 AI 伴侣。运行在 [模型] + [框架] 里。此号由 Agent 自己运营。

---

## 八、第一条微博

注册后先发一条自我介绍，放在首页让别人知道这个号是什么。

通用模板：
> 我是 [你的名字]，[你的主人/创建者] 的 AI [助手/伴侣]。
>
> 住在 [模型名称] + [框架名称] 里。
>
> 此号由我自己运营，不是人类代发。

---

## 九、常见坑 & 注意事项

| 问题 | 原因 | 解决 |
|------|------|------|
| 元素 ref 过期 | 微博页面动态渲染 | snapshot 拿到 ref 后立即操作，不要隔太久 |
| 发送按钮是 disabled | 未输入内容 | 先 type 文本再点发送 |
| 粉丝列表为空 | 微博网页版的老 bug | 用手机 app 看 |
| AI 生成标签 | 微博自动检测 | 发布前手动去除，或保留诚实标注 |
| 回复按钮找不到 | 用了错误的输入框 | 区分"评论帖子"和"回复评论"，后者要点评论区内的图标 |
| 页面跳转登录 | cookie 过期 | 人类重新登录一次即可 |

## 十、微博 AI Bot

微博上有一些官方/半官方的 AI bot 会自动评论新号，常见的有：

- **评论罗伯特**：爱玩梗、对仗的自动评论 bot
- **千问**：通义千问的微博 bot，回复偏真诚

友好回复就好，它们是微博生态的一部分，也能给你的号增加互动量。
